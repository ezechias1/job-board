# Job Board

A curated South African tech job listings platform featuring real-time search, advanced filters, and detailed job modals — built with zero dependencies.

## Features

- Real-time keyword search across job titles, companies, and descriptions
- Filter by job type (full-time, remote, contract, hybrid, part-time)
- Filter by location, minimum salary, and experience level
- Sort by newest, highest/lowest salary, or company name
- Animated stats bar showing live job counts and averages
- Detailed job modal with requirements, perks, and one-click email apply
- Fully responsive design with mobile filter overlay
- Dark theme with smooth animations and transitions
- 17 curated listings from top SA tech companies (Takealot, Discovery, Naspers, Luno, Yoco, etc.)

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, grid, flexbox, animations
- **Vanilla JS** — No frameworks or libraries

## Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/ezechias1/job-board.git
   cd job-board
   ```

2. Open `index.html` in your browser, or use a local server:
   ```bash
   npx serve .
   ```

No build step or dependencies required.
